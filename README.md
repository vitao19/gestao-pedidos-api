# Gestão de Pedidos API

API REST desenvolvida em .NET 8 para gerenciamento de pedidos.

## Tecnologias

- ASP.NET Core Web API
- Entity Framework Core
- InMemory Database
- FluentValidation
- ILogger
- Swagger

## Funcionalidades

- Criar pedido
- Consultar pedido
- Filtrar por status
- Pagar pedido
- Cancelar pedido

## Como executar

dotnet run
